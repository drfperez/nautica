Més endavant trobareu altres d'interès per al projecte escolar 22 dies en 22 peus.
Recordeu que en una de les meves web https://drfperez.github.io hi ha una secció de nàutica amb tots els codis en Javascript adequats pel món web, aquí tindreu les versions Python per resoldre problemes de navegació.
